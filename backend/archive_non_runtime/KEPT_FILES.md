# KEPT FILES — ملفات يحتاجها المشروع أثناء التشغيل

هذا الملف يشرح باختصار الملفات والمجلدات التي بُقيت في جذر المشروع لأنها لازمة لتشغيل التطبيق.

- `app.py`: نقطة دخول التطبيق (Flask) — ضروري.
- `requirements.txt`: تبعيات Python اللازمة للتشغيل.
- `database.py`: وظائف الوصول إلى قاعدة البيانات.
- `models.py`: نماذج التطبيق وعمليات المصادقة.
- `forms.py`: نماذج WTForms المستخدمة في الواجهات.
- `templates/`: قوالب Jinja2 لواجهات الويب.
- `scanners/`: مجلد الماسحات البرمجية; يحتوي الآن على `netscan_scanner.py`, `web_scanner.py`, `sast_scanner.py`, وغيرها — تُستخدم أثناء الفحص.
- `migrations/sql/`: سكريبت إنشاء/هجرة قاعدة البيانات.
- `static assets (css/, js/)`: ملفات الواجهة الأمامية.
- `risk_engine.py`, `server_*` و`dep_scanner.py`, `dast_scanner.py`: محركات التقييم والماسحات المستخدمة أثناء التشغيل.

إذا رغبت، أستطيع توليد نسخة أوامر لتشغيل التطبيق محلياً هنا.
