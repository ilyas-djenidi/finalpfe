====================================================================
1) OBJECTIF DU PROJET
====================================================================
CyBrain est une plateforme web de security assessment qui centralise:
- la soumission de scans de securite
- l'analyse de plusieurs surfaces (reseau, web, serveur, dependances, SAST, DAST)
- le scoring de risque
- la conservation des rapports
- la gestion des utilisateurs et de l'audit

Le projet combine:
- un backend Flask (routes, logique metier, auth, persistance)
- des scanners Python (Nmap, checks web, checks config, Bandit, Nikto, etc.)
- une base SQLite (rapports + audit + users)
- une interface HTML/CSS/JS

3) FLUX DE FONCTIONNEMENT (METHODOLOGIE)
====================================================================
Flux principal:
1. L'utilisateur s'authentifie via /login.
2. Il ouvre / (console de scan) et soumet le formulaire /start-scan.
3. app.py valide les donnees (forms.py), puis choisit le scanner selon scan_type.
4. Le scanner retourne un dict standardise contenant vulnerabilities + meta.
5. risk_engine.calculate_risk() calcule un score 0..10.
6. database.store_report() stocke resultat + score + compteurs + details.
7. L'utilisateur consulte /report/<token> et /dashboard.
8. Toutes les actions sensibles sont tracees dans audit_log.

Principes de securite appliques:
- CSRF active via Flask-WTF
- protection login/session via Flask-Login
- rate limiting via Flask-Limiter
- controle d'acces par permission (decorateur require_permission)
- checks anti-IDOR sur les routes de rapport

====================================================================
15) RESUME EXECUTIF
====================================================================
CyBrain est une plateforme de gouvernance securite orientee operationnel:
- elle orchestre plusieurs moteurs de scan,
- normalise leurs resultats,
- calcule un risque,
- historise les rapports,
- et fournit une couche admin/audit.

Le projet est deja exploitable. La prochaine etape naturelle est la
factorisation du code du scanner web et l'extension de la couverture de tests.
