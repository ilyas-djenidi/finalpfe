# 🚀 CyBrain - دليل التشغيل السريع

## طرق التشغيل

### 1️⃣ **الطريقة الأسهل - النقر المزدوج** ⭐
```
ابحث عن ملف: run.bat
انقر عليه نقراً مزدوجاً
```
- يفعّل البيئة الافتراضية تلقائياً
- يثبت المتطلبات تلقائياً
- يفتح المتصفح تلقائياً
- يشغّل التطبيق على http://127.0.0.1:5000

---

### 2️⃣ **استخدام PowerShell** (خيار بديل)
```powershell
cd e:\PFE\cybrainV4.1\cybrain4.1
.\start.ps1
```

---

### 3️⃣ **التشغيل اليدوي من Terminal**
```bash
cd e:\PFE\cybrainV4.1\cybrain4.1
E:\PFE\.venv\Scripts\activate.bat
pip install -r requirements.txt
python app.py
```

ثم افتح المتصفح على:
```
http://127.0.0.1:5000
```
